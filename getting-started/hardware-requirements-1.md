# Hardware Requirements

## Hardware Requirements

Welcome to the **WizzTakeaway Hardware Requirements** guide.  
Having the right hardware setup is essential for smooth order management and reliable daily operations.

Below are our updated recommendations.

---

## Stable Internet Connection

A stable internet connection is required for the WizzTakeaway cloud-based system.

**Best practices:**

* **Dedicated Internet:** Use a dedicated broadband connection where possible.
* **Backup Connection:** Keep a 4G/5G mobile connection available as a backup (hotspot).
* **Primary 5G Option:** In areas with strong coverage, a 5G connection can be used as the main connection.

{% hint style="warning" %}
**Firewall Settings:**  
Ensure that **wizztakeaway.com** and related services are allowed through your firewall to avoid connectivity issues.
{% endhint %}

## Printers

{% hint style="info" %}
Printers are optional but **highly recommended** for kitchen and order management.
Thermal receipt printers offer the best performance.
{% endhint %}

### Printing from Windows

Any printer that works with Windows (Word / Notepad printing works) is compatible.

### Printing from Android

Use **ESCPOS-compatible printers** connected via:

* Ethernet
* Wi-Fi
* Bluetooth

---

## Recommended Printers

We recommend **PushPrinter** devices due to reliability and integration support.

**Recommended Models:**

* PushPrinter PX-90 (Ethernet or Bluetooth)
* Epson TM-T20 (USB / Ethernet)
* Epson TM-M30 (Bluetooth)
* Epson TM-T82-II-i (Multi-device support)

{% hint style="warning" %}
**Android Users:**  
Star Micronics printers are **not recommended** for Android devices.
{% endhint %}

## Frequently Asked Questions (FAQ)

**Q: Can I manage orders using a mobile phone?**  
A: Yes, but we strongly recommend a tablet or computer for daily operations.

**Q: What if my internet connection fails?**  
A: Use a mobile hotspot (4G/5G) as a backup connection.
